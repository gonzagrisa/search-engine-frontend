Páginas

Una de las posiblidades para indexar sitios web es el de cargar las paginas manualmente, para esto tiene que ingresar a su cuenta e ir a la sección de páginas para empezar a cargar las páginas manualmente, cabe aclarar que si la página no se encuentra disponible al momento de la inserción, la misma no podrá ser ingresada para indexación. Además no podrá poner 2 páginas que pertenezcan al mismo dominio, ya que los resultados de indexación estarían repetidos.

Si usted tiene un servicio registrado para el listado de páginas, y una de esas páginas es ejemplo.com, si ejemplo.com ya fue ingresada manualmente pero no fue indexada, es decir, su estado es 'A Indexar', esa página pasará a ser parte del servicio.

Servicios

DogDogGo también permite que las paginas a indexar provengan desde un servicio. Para esto, debe proveernos de la url del recurso, y nuestro sistema se encargará de buscar los endpoints correspondientes, uno que es el endpoint de ping de su servicio y otro que es el endpoint que contiene la lista de sitios web que desea indexar.
Se permite el uso de tanto servicios REST como servicios web (SOAP), cada uno debe cumplir las siguientes condiciones para que puedan ser consumidos y utlizados de forma correcta:


REST

Para el consumo correcto de su servicio REST, debe proveernos sólo del link del servicio, el mismo deberá contar con 2 endpoints (ping y list), nuestro sistema se encargará de agregar los endpoints de ping y list como se muestra a continuación, suponiendo que http://localhost:/8080/api/ es la URL de su servicio.

Estados

En el portal, podrá verá los siguientes estados:

A Indexar

Este estado indica que el servicio o página están marcados para ser indexados o reindexados en la próxima ejecución del crawler

Indexado

Este estado indica que el servicio o página ya fueron indexados, lo que significa que los metadatos generados a partir de ellos ya están disponibles para editar, eliminar o actualizar o que ya están disponibles para la búsqueda

Caído

Este estado indica que el servicio o página no está disponible, no pudiendo consumirlo para crawlear la página definida u obtener el listado de sitios web a partir del servicio
